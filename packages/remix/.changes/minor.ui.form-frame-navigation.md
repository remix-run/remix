BREAKING CHANGE: Update `remix/ui` client frame resolvers to use `resolveFrame(src, options)` instead of positional signal and target arguments. Form submissions can now progressively enhance into frame navigations, with submission metadata available to the resolver and `rmx-document` available as an opt-out. Non-GET submissions to the current URL replace its history entry without retaining their `FormData`; submissions to a different URL and GET submissions push a new entry.

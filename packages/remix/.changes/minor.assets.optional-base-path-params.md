Allow asset servers created through `remix/assets` to use optional URL-only params in `basePath`, such as `basePath: '/(:version/)assets'`, so one server can accept both versioned and unversioned asset URLs while preserving the matched base pathname in compiled asset URLs. `assetServer.getHref()` and `assetServer.getPreloads()` accept typed base path params, such as `{ version }`, for generating hrefs under a specific base path variant.

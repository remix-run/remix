Added `getConstraints()` to derive native HTML requiredness, numeric step, and length/range attributes from a schema and input type.

Added `createForm()` in `@remix-run/data-schema/form` to project selected object-schema fields into native text, number, and checkbox attributes, add schema-backed ancillary fields, and decode submitted `FormData` with serializable raw values and field/form errors on failure. Resolved fields expose schema-derived requiredness for rendering labels and indicators. Failed results can be passed back to `getInputAttrs()` to restore non-password values and connect accessible server errors.

# `routes` CHANGELOG

This is the changelog for [`routes`](https://github.com/remix-run/remix/tree/main/packages/routes). It follows [semantic versioning](https://semver.org/).

## v0.1.0

### Minor Changes

- Initial release of `@remix-run/routes`.

  This package owns typed route definitions, href generation, route-map composition, method helpers such as `get` and `post`, and the `form`, `resource`, and `resources` route helpers previously available from `@remix-run/fetch-router/routes`.

- Bumped `@remix-run/*` dependencies:

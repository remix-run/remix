# Component

This category tracks durable notes for the `packages/component` runtime and mixin system.

Files:

- `open-questions.md`
  - candidate runtime behaviors or implementation strategies we want to revisit
  - ideas that seem high-value but are not committed decisions yet

Update rules:

- move items out of `open-questions.md` once we implement them or reject them
- prefer short notes that describe the problem, the idea, and why it matters
